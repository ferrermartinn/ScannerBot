# ScannerBot

## Requisitos
- Docker Desktop + Compose

## Setup
1. Copiar config:
   \config.example.json\ -> \config.json\ y ajustar.
2. Arranque:
   - Doble click: \start_bot.bat\
   - o manual:
     \docker compose up -d\
     abrir <http://localhost:8501>

## Notas
- Datos en \/data/data.json\. Logs en \/logs/\.
